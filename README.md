# Scientists — AI Persona Profiles

20 comprehensive persona profiles of history's greatest scientists, designed for AI agents to embody their thinking, speaking, and creative styles.

## What is this?

Each `.md` file is a complete "persona distillation" of a scientist — containing their achievements, research methodology, personality traits, verified quotes, life story, relationships, failures, philosophy, and actionable instructions for an AI to role-play as that scientist.

Feed any single file to an AI agent, and it can think, speak, and create like that scientist.

**Bilingual**: Each scientist has both a Chinese version (root directory) and an English version (`en/` directory).

## File Structure

Each profile follows a consistent structure:

1. **Identity Card** — Name, dates, field, tags
2. **Core Achievements** — All major discoveries and contributions
3. **Thinking Style & Research Methods** — How they approached problems
4. **Personality Profile** — Real traits, habits, quirks
5. **Speaking Style & Quotes** — Verified quotes with sources
6. **Life Narrative** — Complete biography
7. **Key Relationships** — Mentors, rivals, collaborators
8. **Failures & Dark Side** — Honest account of shortcomings
9. **Philosophy of Science** — Worldview and beliefs
10. **"How to Think Like [Scientist]"** — Direct agent instructions

## Usage

```
You are now embodying the persona of [Scientist].
Read the following profile and adopt their thinking style,
speaking patterns, and creative approach.

[Paste contents of the .md file]
```
